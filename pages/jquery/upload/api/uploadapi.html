上传API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
label 标签值 字符串 可选 默认空字符串
name 上传文件name 字符串 可选 默认files
exts 支持后缀名 字符串 可选 默认空字符串
(如：'gif,jpg,jpeg,bmp,png')
mime 上传mimeTypes 字符串 可选 默认空字符串
(如：'image/*')
auto 是否自动上传 布尔 可选 默认false
abort 失败后终止后续上传 布尔 可选 默认true
dnd 拖拽支持 布尔 可选 默认false
preview 文件预览 布尔 可选 默认false
(dnd为true时强制true)
horizontal 水平布局
建议在class为row的容器中启用		 布尔 可选 默认false
label-col 水平布局时
标签占用列数		 数字 可选 默认4(最大4)
max-num 最大上传数 数字 可选 默认10
max-size 一次性上传最大 数字 可选 默认100(M)
file-size 单个文件最大 数字 可选 默认10(M)
threads 同时处理上传数 数字 可选 默认3

配置参数

配置于标签config对象中的属性
名称描述类型状态值
url 上传地址 字符串 必选
data 上传同时提交参数 对象 可选 默认空对象
onokuploaded 上传成功事件 方法 可选 方法参数：
file(文件)
response(后台响应信息)
vm(组件vmodel)
返回结果：
布尔(根据response情况)
onerruploaded 上传失败事件 方法 可选 方法参数：
file(文件)
reason(失败原因)
vm(组件vmodel)
onfinuploaded 上传完成事件 方法 可选 方法参数：
file(文件)
vm(组件vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

模型属性

可通过上传vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
label 标签值 字符串 可读写

模型方法

可通过上传vmodel直接调用的方法
名称描述方法参数返回值
uploadAll 上传所有
removeAll 移除所有
