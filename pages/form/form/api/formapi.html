表单API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
title 表单标题 字符串 可选 默认空字符串
btn-size 按钮大小 字符串 可选 默认'normal'(可选'small')
btn-pos 按钮位置 字符串 可选 默认'left'(可选'right')
submit-mode 提交模式 字符串 可选 默认'ajax'(可选'form')
border 边框模式 布尔 可选 默认false
buttons 是否带提交/重置按钮 布尔 可选 默认true

配置参数

配置于标签config对象中的属性
名称描述类型状态值
submitUrl 提交请求地址 字符串 可选 默认空字符串
loadUrl 请求数据地址 字符串 可选 默认空字符串
loadParam 请求数据参数 对象 可选 默认空对象
{"rspcod": "200", "rspmsg": "查询成功", "data": 数据对象}
onloaded 数据加载成功事件 方法 可选 方法参数：
dat(数据对象), vm(表单vmodel)
onsubmited 点击提交按钮事件
调用提交方法事件		 方法 可选 方法参数：
ev(事件对象), vm(表单vmodel)
onreseted 点击重置按钮事件
调用重置方法事件		 方法 可选 方法参数：
ev(事件对象), vm(表单vmodel)
onoksubmited 表单提交成功事件 方法 可选 方法参数：
dat(数据对象), vm(表单vmodel)
onerrsubmited 表单请求异常事件 方法 可选 方法参数：
dat(异常对象), vm(表单vmodel)

插入片段(slot)

在表单标签中插入的html片段
名称描述类型状态值
content 表单内容 html/tdui 可选 默认空字符串

模型属性

可通过表单vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
title 表单标题 字符串 可读写

模型方法

可通过表单vmodel直接调用的方法
名称描述方法参数返回值
setData 设置表单内容
当前仅可设置tdui		 data:数据对象
key为元素name		 无
getData 获取表单内容
当前仅可获取tdui		 无 数据对象
key为元素name
reload 重新加载数据 无 无
submit 提交表单 无 无
reset 重置表单 无 无
getElements 获取表单元素
当前仅可获取tdui		 无 对象数组
