评分API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
label 标签 字符串 可选 默认空字符串
name 控件名 字符串 可选 默认'rate'
value 控件值 数字 可选 默认0
num 最大分数 数字 可选 默认5
disabled 是否不可用 布尔 可选 默认false
half 是否可半分 布尔 可选 默认false

配置参数

配置于标签config对象中的属性
名称描述类型状态值
onclicked 点击事件 方法 可选 方法参数：
ev(事件对象), vm(控件vmodel)
onchanged 改变事件 方法 可选 方法参数：
ev(事件对象), vm(控件vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

模型属性

可通过评分vmodel直接调用的属性
名称描述类型状态
disabled 是否禁用 布尔 可读写
其他标签属性 其他标签属性 建议只读

模型方法

可通过评分vmodel直接调用的方法
名称描述方法参数返回值
setValue 设置值 val:数字 无
getValue 获取值 无 数字
getData 获取对象值 无 name:value对象
