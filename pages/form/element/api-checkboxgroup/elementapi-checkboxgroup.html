复选框组API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
label 标签 字符串 可选 默认空字符串
name 控件名 字符串 可选 默认'checkboxgroup'

配置参数

配置于标签config对象中的属性
名称描述类型状态值
checkboxes 复选框组 对象数组 必选
checkboxes元素详细
label 复选框标签 字符串 必选
value 复选框值 字符串 必选
checked 是否选中 布尔 必选
disabled 是否不可用 布尔 必选
onchecked 选中事件 方法 可选 方法参数：ev(事件对象),vm(控件vmodel),box(复选框对象)
onchanged 改变事件 方法 可选 方法参数：ev(事件对象),vm(控件vmodel),box(复选框对象)
onchanged 组改变事件 方法 可选 方法参数：ev(事件对象),vm(控件vmodel)

模型属性

可通过复选框组vmodel直接调用的属性
名称描述类型状态
checkboxes 复选框组 对象数组 建议只读(元素属性可读写)

模型方法

可通过复选框组vmodel直接调用的方法
名称描述方法参数返回值
setValue 设置值 val:字符串 无
getValue 获取值 无 字符串
getData 获取对象值 无 name:value对象
