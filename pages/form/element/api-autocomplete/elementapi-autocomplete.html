自动补全API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
label 标签 字符串 可选 默认空字符串
name 控件名 字符串 可选 默认'autocomplete'
disabled 是否不可用 布尔 可选 默认false
must 是否必选 布尔 可选 默认false
horizontal 水平布局
建议在class为row的容器中启用		 布尔 可选 默认false
label-col 水平布局时
标签占用列数		 布尔 可选 默认4(最大4)

配置参数

配置于标签config对象中的属性
名称描述类型状态值
data 补全数据 对象 可选 默认空对象
url 请求补全数据 字符串 可选 默认空字符串
param 请求补全参数 对象 可选 默认空对象
{"rspcod": "200", "data": 补全数据对象}
onloaded 数据加载事件 方法 可选 方法参数：
dat(数据对象), vm(控件vmodel)
onselected 选中事件 方法 可选 方法参数：
ev(事件对象), vm(控件vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

模型属性

可通过自动补全vmodel直接调用的属性
名称描述类型状态
disabled 是否禁用 布尔 可读写
其他标签属性 其他标签属性 建议只读

模型方法

可通过自动补全vmodel直接调用的方法
名称描述方法参数返回值
getText 获取文本值 无 字符串
getValue 获取key值 无 字符串
getData 获取对象值 值类型参数
true为key值,false为文本值		 对象({name:xxx})
