数据表格API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
title 表格标题 字符串 可选 默认空字符串
operation 下拉按钮组名 字符串 可选 默认'操作'
btn-name 工具条默认按钮名 字符串 可选 默认'查询'
btn-icon 工具条默认按钮图标 字符串 可选 默认'glyphicon glyphicon-search'
height 表格高度 数字 可选 默认'auto'
limit 页大小 数字 可选 默认10
auto 自动加载数据 布尔 可选 默认false
checkbox 是否带复选框 布尔 可选 默认true
singleSelect 是否单选 布尔 可选 默认false
editable 双击编辑 布尔 可选 默认false
bar 是否带标题工具栏
(false时无标题)		 布尔 可选 默认true

配置参数

配置于标签config对象中的属性
名称描述类型状态值
cols 表格列模型 对象数组 必选
cols元素详细
name 列标识 字符串 必选
display 列名称 字符串 必选
width 列宽 数字 可选
render 渲染单元格 方法 可选 方法参数：
val(单元格内容)
row(当前行数组)
ridx(当前行索引)
cidx(当前列索引)
方法返回：
渲染后的单元格HTML
fun 点击单元格事件 方法 可选 方法参数：
ev(事件对象)
vm(表格vmodel)
row(行索引)
col(列索引)
val(单元格值)
disabled 列禁止编辑 布尔 可选 默认false
type 列类型 字符串 必选 默认'text'('select')
option 列选项 对象 必选 数据结构{k:v}
rows 行数据 对象数组 可选
rows元素详细
各列name 列name对应数据 字符串 必选
selected 是否选中 布尔 必选
loadUrl 请求数据地址 字符串 可选 默认空字符串
loadParam 请求数据参数 对象 可选 默认空对象
{"total":"5","rspcod" :"200","rspmsg":"","rows":行数据rows}
updateUrl 双击行更新地址 字符串 可选 默认空字符串
deleteUrl 删除数据地址 字符串 可选 默认空字符串
提交参数:deleteKey
key 行主键 字符串数组 存在deleteUrl时必选 默认空数组
用于后台删除的主键
buttons 工具条按钮组 对象数组 可选 默认空数组
buttons元素详细
display 按钮显示 字符串 必选
icon 按钮图标 字符串 可选 图标详情
fun 按钮点击事件 方法 可选 方法参数：
ev(事件对象)
vm(表格vmodel)
operations 下拉按钮组 对象数组 可选 默认空数组
operations元素详细
display 按钮显示 字符串 必选
icon 按钮图标 字符串 可选 图标详情
fun 按钮点击事件 方法 可选 方法参数：
ev(事件对象)
vm(表格vmodel)
actions 浮动按钮组 对象数组 可选 默认空数组
actions元素详细
title 按钮title属性 字符串 可选 默认空字符串
icon 按钮图标 字符串 可选 图标详情
type 按钮类型 字符串 可选 默认空字符串('primary'/'info'/'success'/'warning'/'danger'/'default')
fun 按钮点击事件 方法 可选 方法参数：
ev(事件对象)
vm(表格vmodel)
render 渲染响应结果
如：汇总信息		 方法 可选 方法参数：
vm(表格vmodel)
dat(响应数据对象)
返回结果：
html字符串
onloaded 请求数据
完成事件		 方法 可选 方法参数：
dat(数据对象)
vm(表格vmodel)
onreloaded 重新请求数据
完成事件		 方法 可选 方法参数：
dat(数据对象)
vm(表格vmodel)
onrowclicked 行点击事件 方法 可选 方法参数：
row(行对象)
vm(表格vmodel)
onrowdbclicked 行双击事件 方法 可选 方法参数：
row(行对象)
vm(表格vmodel)
onrowselected 行选中事件 方法 可选 方法参数：
row(行对象)
vm(表格vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

插入片段(slot)

在数据表格标签中插入的html片段
名称描述类型状态值
content 查询条件位置内容 html/tdui 可选 默认空字符串

模型方法

可通过数据表格vmodel直接调用的方法
名称描述方法参数返回值
loadData 继续加载数据 p:加载参数对象 无
reloadData 重新加载数据 p:加载参数对象 无
getSelectedIdx 获取选中行索引 无 数字数组
getSelectedRow 获取选中行对象 无 对象数组
removeRow 删除指定行 行索引数组 删除行数
removeSelectedRow 删除选中行 无 删除行数
modifyRow 修改指定行 行索引数
修改数据对象		 无
modifySelectRow 修改选中行
多选只修改第一行		 修改数据对象 无
addRow 添加行 行数据对象数组 无
getRow 获取指定行 行索引数组 对象数组
