树API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
checkbox 是否带复选框 布尔 可选 默认false
root 是否带root样式 布尔 可选 默认true

配置参数

配置于标签config对象中的属性
名称描述类型状态值
data 树节点数据 对象数组 可选 默认空数组
data元素详细
id 节点ID 字符串 必选
text 节点名称 字符串 必选
expand 是否展开 布尔 必选
checked 是否选中 布尔 checkbox为true时必选
disabled 是否不可用 布尔 必选
icon 节点图标 字符串 可选 图标详情
href 节点链接目标 字符串 可选 默认'javascript:void(0)'
children 子节点 对象数组 必须 同节点数据data
url 请求节点数据地址 字符串 可选 默认空字符串
param 请求节点数据参数 对象 可选 默认空对象
{"rspcod":"200","rspmsg":"","data": 节点数据data}
onloaded 数据加载完成事件 方法 可选 方法参数：
dat(请求数据),vm(树vmodel)
onclicked 节点点击事件 方法 可选 方法参数：
el(节点对象),vm(树vmodel)
onchecked 节点选中事件 方法 可选 方法参数：
el(节点对象),vm(树vmodel)
oncollapsed 节点收起事件 方法 可选 方法参数：
el(节点对象),vm(树vmodel)
onexpanded 节点展开事件 方法 可选 方法参数：
el(节点对象),vm(树vmodel)

模型方法

可通过树vmodel直接调用的方法
名称描述方法参数返回值
getChecked 获取选中节点ID 无 以逗号分隔的ID列表
reloadData 重新加载节点数据 无 无
