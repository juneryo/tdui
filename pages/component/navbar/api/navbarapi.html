导航条API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
title 导航条标题 字符串 可选 默认空字符串
operation 下拉操作名称 字符串 可选 默认'操作'
form 是否有表单文本框 布尔 可选 默认false
href 标题链接目标 字符串 可选 默认'javascript:void(0)'
margin 底部外补丁 数字 可选 默认20
padding 左右内补丁 数字 可选 默认0

配置参数

配置于标签config对象中的属性
名称描述类型状态值
buttons 导航按钮组 对象数组 可选 默认空数组
buttons元素详细
display 按钮名称 字符串 必选
icon 按钮图标 字符串 可选 图标详情
href 按钮链接目标 字符串 可选 默认'javascript:void(0)'
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(导航条vmodel)
operations 下拉按钮组 对象数组 可选 默认空数组
operations元素详细
display 按钮名称 字符串 必选
icon 按钮图标 字符串 可选 图标详情
href 按钮链接目标 字符串 可选 默认'javascript:void(0)'
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(导航条vmodel)
ontitleclicked 点击标题事件 方法 可选 方法参数：
ev(事件对象), vm(导航条vmodel)
onsubmited 文本框回车提交事件
form需为true		 方法 可选 方法参数：
ev(事件对象), vm(导航条vmodel)

模型属性

可通过导航条vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
title 导航条标题 字符串 可读写
operation 下拉操作名 字符串 可读写
buttons 导航按钮组 对象数组 建议只读
operations 下拉按钮组 对象数组 建议只读

模型方法

可通过导航条vmodel直接调用的方法
名称描述方法参数返回值
setTitle 设置导航条标题 title:字符串 无
getTitle 获取导航条标题 无 导航条标题字符串
setOperation 设置下拉操作名 oper:字符串 无
getOperation 获取下拉操作名 无 名称字符串
