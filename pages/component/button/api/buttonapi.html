按钮API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
value 按钮值 字符串 可选 默认空字符串
type 按钮类型 字符串 可选 默认空字符串('primary'/'info'/'success'/'warning'/'danger'/'default')
size 按钮尺寸 字符串 可选 默认空字符串('large'/'normal'/'small'/'xsmall')
badge 徽章值 字符串 可选 默认空字符串
disabled 是否禁用 布尔 可选 默认false

配置参数

配置于标签config对象中的属性
名称描述类型状态值
actions 下拉按钮列表 对象数组 可选 默认空数组
actions元素详细
display 按钮名称 字符串 必选
type 按钮类型 字符串 可选 默认空字符串('split')
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(按钮vmodel)
onclicked 按钮点击事件 方法 可选 方法参数：
ev(事件对象), vm(按钮vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

模型属性

可通过按钮vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
value 按钮值 字符串 可读写
type 按钮类型 字符串 可读写
size 按钮尺寸 字符串 可读写
badge 徽章内容 字符串 可读写
disabled 是否禁用 布尔 可读写
actions 下拉按钮列表 对象数组 建议只读

模型方法

可通过按钮vmodel直接调用的方法
名称描述方法参数返回值
click 触发按钮点击 无 无
