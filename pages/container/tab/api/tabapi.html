标签页API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
active 当前展现标签索引 数字 可选 默认0
padding 标签页内补丁 数字 可选 默认15
anime 动画效果(IE10+) 字符串 可选 默认normal(可选cool)

配置参数

配置于标签config对象中的属性
名称描述类型状态值
tabs 标签组 对象数组 必选
tabs元素详细
title 标签名称 字符串 必选
fun 标签点击回调 方法 可选 方法参数：
obj(newVal:新展开索引,oldVal:旧索引)
vm(标签页vmodel)
onchanged 标签改变事件 方法 可选 方法参数：
obj(newVal:新标签索引,oldVal:旧索引)
vm(标签页vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

插入片段(slot)

在标签页标签中插入的html片段
名称描述类型状态值
content0 第1个标签内容 html/tdui 必选 默认空字符串
contentn 第n个标签内容
n为标签数		 html/tdui 必选 默认空字符串

模型属性

可通过标签页vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
active 当前标签索引 数字 可读写
tabs 标签组 对象数组 建议只读

模型方法

可通过标签页vmodel直接调用的方法
名称描述方法参数返回值
setActive 设置显示标签索引 idx:数字 无
getActive 获取显示标签索引 无 当前显示标签索引
