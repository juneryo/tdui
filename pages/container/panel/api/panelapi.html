面板API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
title 面板标题 字符串 可选 默认空字符串
operation 下拉操作名 字符串 可选 默认'操作'

配置参数

配置于标签config对象中的属性
名称描述类型状态值
buttons 工具栏按钮 对象数组 可选 默认空数组
buttons元素详细
display 按钮名称 字符串 必选
icon 按钮图标 字符串 可选 图标详情
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(面板vmodel)
operations 工具栏下拉按钮 对象数组 可选 默认空数组
operations元素详细
display 按钮名称 字符串 必选
icon 按钮图标 字符串 可选 图标详情
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(面板vmodel)
onready 组件就绪事件 方法 可选 方法参数：
elem(组件html元素)
vm(组件vmodel)

插入片段(slot)

在面板标签中插入的html片段
名称描述类型状态值
content 面板体内容 html/tdui 可选 默认空字符串
footer 面板底部内容 html/tdui 可选 默认空字符串

模型属性

可通过面板vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
title 面板标题 字符串 可读写
buttons 面板按钮组 对象数组 建议只读

模型方法

可通过面板vmodel直接调用的方法
名称描述方法参数返回值
setTitle 设置面板标题 title:字符串 无
getTitle 获取面板标题 无 面板标题字符串
