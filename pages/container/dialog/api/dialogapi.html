弹出框API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
title 弹出框标题 字符串 可选 默认空字符串
show 弹出框是否显示 布尔 可选 默认false
size 弹出框尺寸 字符串 可选 默认'normal'
('large'/'small')
padding 弹出框体内补丁 数字 可选 默认15

配置参数

配置于标签config对象中的属性
名称描述类型状态值
buttons 底部按钮组 对象数组 可选 默认空数组
buttons元素详细
display 按钮名称 字符串 必选
type 按钮类型 字符串 可选 默认空字符串('primary'/'info'/'success'/'warning'/'danger'/'default')
fun 按钮点击回调 方法 可选 方法参数：
ev(事件对象), vm(弹出框vmodel)
onshowed 显示事件 方法 可选 方法参数：
obj(空对象), vm(弹出框vmodel)
onhided 隐藏事件 方法 可选 方法参数：
obj(空对象), vm(弹出框vmodel)

插入片段(slot)

在弹出框标签中插入的html片段
名称描述类型状态值
content 面板体内容 html/tdui 可选 默认空字符串

模型属性

可通过弹出框vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
title 弹出框标题 字符串 可读写
show 弹出框是否显示 布尔 可读写
buttons 弹出框底部按钮组 对象数组 建议只读

模型方法

可通过弹出框vmodel直接调用的方法
名称描述方法参数返回值
setTitle 设置弹出框标题 title:字符串 无
getTitle 获取弹出框标题 无 弹出框标题字符串
showDialog 显示弹出框 无 无
hideDialog 隐藏弹出框 无 无
