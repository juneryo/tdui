折叠面板API


标签属性

存在于标签内部的属性
名称描述类型状态值
$id 唯一标识 字符串 可选 默认随机生成
config 模型配置 对象 可选 默认空对象
active 当前展开面板索引 数字 可选 默认0(从0开始)
padding 面板体内补丁 数字 可选 默认15

配置参数

配置于标签config对象中的属性
名称描述类型状态值
panels 面板组 对象数组 必选
panels元素详细
title 面板名称 字符串 必选
fun 面板点击回调 方法 可选 方法参数：
obj(newVal:新展开索引,oldVal:旧索引)
vm(折叠面板vmodel)
onchanged 折叠改变事件 方法 可选 方法参数：
obj(newVal:新展开索引,oldVal:旧索引)
vm(折叠面板vmodel)

插入片段(slot)

在折叠面板标签中插入的html片段
名称描述类型状态值
content0 第1个面板内容 html/tdui 必选 默认空字符串
contentn 第n个面板内容
n为面板数		 html/tdui 必选 默认空字符串

模型属性

可通过折叠面板vmodel直接调用的属性(所有属性都建议只读)
名称描述类型状态
active 当前展开面板索引 数字 可读写
panels 面板组 对象数组 建议只读

模型方法

可通过折叠面板vmodel直接调用的方法
名称描述方法参数返回值
setActive 设置展开面板 idx:数字 无
getActive 获取展开面板索引 无 当前展开面板索引
